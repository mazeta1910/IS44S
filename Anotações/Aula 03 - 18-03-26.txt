KHTML é um motor de renderização criado para o navegador Konqueror, do ambiente Linux/KDE. Ele interpretava código HTML e CSS para mostrar páginas.
Resumindo: KHTML foi o avô técnico dos motores que hoje movem Safari, Chrome, Edge e outros navegadores.
